# Fibonacci Bot — Bybit Futures

Торговый бот по уровням Фибоначчи. Таймфрейм 15м, плечо ×10.

## Структура файлов

```
main.py              ← точка входа, запускай отсюда
backtest.py          ← бэктест на исторических данных
telegram_notify.py   ← уведомления в Telegram
requirements.txt     ← зависимости
bot.log              ← лог работы бота (создаётся автоматически)
backtest_results.json← результаты бэктеста (создаётся автоматически)
```

## Установка

```bash
pip install -r requirements.txt
```

## Настройка (main.py)

| Параметр | Что делает |
|---|---|
| `API_KEY / API_SECRET` | Ключи от Bybit (только фьючерсные права) |
| `TESTNET = True` | Тестнет / реальный аккаунт |
| `SYMBOLS` | Список монет |
| `LEVERAGE` | Плечо (по умолчанию 10) |
| `RISK_PCT` | Риск на сделку в % от депозита |
| `RUN_BACKTEST_FIRST` | Запускать бэктест перед стартом |
| `MIN_BACKTEST_WINRATE` | Минимальный winrate для старта бота |

## Настройка Telegram (telegram_notify.py)

1. Создай бота у @BotFather → получи `TG_TOKEN`
2. Напиши боту любое сообщение, затем узнай `chat_id` у @userinfobot
3. Вставь оба значения в `telegram_notify.py`

## Запуск

```bash
# Тестнет (рекомендуется минимум 2 недели)
python main.py

# После успешного тестнета — поставь TESTNET = False в main.py
```

## Логика входа

1. Находит свинг-хай и свинг-лой за 50 свечей
2. Строит уровни Фибоначчи (0.382 / 0.5 / 0.618 / 0.786)
3. Ждёт касания ± 0.1%
4. Проверяет объём (×1.3 от среднего) и подтверждение свечой
5. Открывает позицию со стопом и тейком

## Защита

- Дневная остановка после 5 убытков подряд
- Один открытый ордер на монету одновременно
- Бот не стартует если бэктест ниже минимального winrate

## ⚠️ Важно

Торговля с плечом несёт высокий риск потери депозита.
Тестируй только на тестнете до получения стабильных результатов.
